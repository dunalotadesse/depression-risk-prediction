# Predicting Risk of Depression Using Real Health Data

## Overview
This project uses a real-world dataset from Kaggle to analyze and predict depression risk using machine learning.

## Objectives
- Explore mental health data
- Identify key risk factors
- Build a predictive model

## Dataset
- Source: Kaggle Mental Health Dataset
- Features include stress level, sleep hours, age, and lifestyle indicators

## Tools Used
- Python
- Pandas, NumPy
- Scikit-learn
- Matplotlib

## Project Workflow
1. Data loading and exploration  
2. Data cleaning  
3. Exploratory Data Analysis (EDA)  
4. Feature encoding  
5. Model training (Logistic Regression)  
6. Model evaluation  

## Results
The model predicts depression risk with reasonable accuracy, showing how data science can support mental health analysis.

##  Social Impact
Mental health is a global challenge. This project highlights how data-driven approaches can support early detection and intervention.
